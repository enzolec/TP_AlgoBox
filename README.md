\# Projet AlgoBox – Collaboration avec GitHub



\## Présentation du projet



Ce projet a été réalisé dans le cadre du module d’algorithmique en 2A à l’EPF.  

L’objectif principal est de développer plusieurs algorithmes en AlgoBox et d’utiliser Git et GitHub pour apprendre à travailler en binôme avec un gestionnaire de versions.



\- \*\*Étudiant :\*\* Leceuve Enzo

\- \*\*Dépôt GitHub :\*\* https://github.com/enzolec/TP\_Algobox



\## Prérequis



Pour ouvrir et exécuter les fichiers :



\- AlgoBox (version 1.1.x ou équivalent), disponible sur le site officiel.

\- Un ordinateur sous Windows (config testée dans notre cas).



## Contenu du dépôt

- `src/` : contient l’ensemble des fichiers `.algo` du projet.  
  Ces fichiers sont principalement des algorithmes de test permettant de prendre en main AlgoBox
  (déclaration de variables, affichage de messages, premières structures de contrôle, etc.).
- `README.md` : ce document d’explication.




\## Comment ouvrir les fichiers AlgoBox



1\. Lancer \*\*AlgoBox\*\*.

2\. Aller dans \*\*Fichier → Ouvrir\*\*.

3\. Naviguer jusqu’au dossier `src/` du projet cloné (`TP\_Algobox`).

4\. Sélectionner le fichier `.algo` souhaité.

5\. Lancer l’algorithme avec le bouton \*\*Exécuter\*\*.
